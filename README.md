# kibana-logo-hack
Kibana Plugin for Logo Neutralization

### Installation
```
bin/kibana plugin --install kibana-logo-hack -u https://github.com/elasticfence/kibana-logo-hack/archive/master.tar.gz
```
#### Usage
```
BRAND="NULL" bin/kibana
```
